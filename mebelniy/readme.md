# BeautyBix - Telegram Bot для бьюти-салона

Бот для записи на процедуры в бьюти-салоне BeautyBix.

## Возможности

- 💄 Просмотр услуг и цен
- 📝 Онлайн-запись на процедуры
- 🎫 Система промокодов и скидок
- ❓ FAQ и консультации
- 📞 Контактная информация

## Установка и запуск

### 1. Установите зависимости

```bash
pip install -r requirements.txt
```

### 2. Настройте переменные окружения

Отредактируйте файл `.env`:

```env
# Настройки бота BeautyBix
BOT_TOKEN=your_bot_token_here

# Контактная информация салона
CONTACT_USERNAME=@beautybix_salon
CONTACT_PHONE=+7 999 123 45 67
ADMIN_CHAT_ID=your_admin_chat_id_here

# Адрес салона
SALON_ADDRESS=ул. Красивая, 123, Москва
SALON_HOURS=Пн-Вс: 9:00 - 21:00

# Промокоды
PROMO_CODE_WELCOME=WELCOME20
PROMO_CODE_FIRST=FIRST15
```

### 3. Получите токен бота

1. Создайте бота через [@BotFather](https://t.me/BotFather)
2. Получите токен и вставьте в `.env` файл
3. Укажите ID чата администратора для получения заявок

### 4. Запустите бота

```bash
python bot.py
```

## Структура проекта

- `bot.py` - основной файл запуска бота
- `config.py` - конфигурация и настройки
- `handlers (1).py` - обработчики сообщений и команд
- `keyboards.py` - клавиатуры и кнопки
- `states.py` - состояния FSM для форм
- `validators.py` - валидация данных
- `utils.py` - вспомогательные функции
- `storage.py` - хранение данных
- `.env` - переменные окружения
- `requirements.txt` - зависимости Python

## Настройка

### Услуги салона

Услуги настраиваются в файле `config.py` в массиве `SERVICES`. Каждая услуга содержит:
- `emoji` - эмодзи для отображения
- `id` - уникальный идентификатор
- `name` - название услуги
- `description` - подробное описание с ценами

### Промокоды

Промокоды настраиваются в `config.py` в словаре `PROMO_CODES`.

### Контактная информация

Все контакты настраиваются в файле `.env`.

## Функции бота

1. **Главное меню** - навигация по разделам
2. **Услуги** - просмотр каталога с ценами
3. **Запись** - форма записи на процедуры
4. **Промокоды** - система скидок
5. **FAQ** - часто задаваемые вопросы
6. **О салоне** - информация о салоне

## Поддержка

Для вопросов и поддержки обращайтесь к администратору салона.
